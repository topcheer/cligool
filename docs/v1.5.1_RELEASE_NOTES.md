# CliGool v1.5.1 - CLI WebSocket 稳定性修复

## 🎯 本次更新重点

**CliGool v1.5.1** 是一次稳定性补丁更新，主要修复 CLI 客户端在长连接与退出阶段的 WebSocket 并发写入问题。

## ✅ 修复内容

### 🔧 修复 CLI 并发写入 WebSocket 导致的 panic
- 修复 `panic: concurrent write to websocket connection`
- 修复在 `--cmd claude` 等直接执行命令场景下更容易触发的问题
- 将 CLI 客户端的文本消息统一改为串行写入，避免多个 goroutine 同时调用 `WriteMessage`

### 💓 修复心跳与退出阶段的写入冲突
- 心跳 `ping/pong` 改为使用 `WriteControl`
- 关闭阶段的 `close` 消息改为通过统一写入器发送
- 避免心跳、普通输出、关闭消息互相抢占同一个 WebSocket 连接

### 🧹 优化退出日志
- 关闭连接时不再把预期内的 `use of closed network connection` 视为异常
- 降低 CLI 退出阶段的噪音日志，便于定位真正的问题
