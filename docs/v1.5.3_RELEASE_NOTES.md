# CliGool v1.5.3 - Windows Web Terminal Echo Fixes

## 🎯 本次更新重点

**CliGool v1.5.3** 重点修复 Windows CLI 场景下 Web 终端重复回显/重复提示符的问题，并统一本地多平台构建脚本行为。

## ✅ 修复内容

### 🪟 修复 Windows Web 终端重复提示符
- 移除浏览器端在初始化后自动注入的 synthetic Enter
- 避免 Windows `cmd/ConPTY` 把空回车当成真实命令执行
- 修复会话初始化或重连后连续打印额外 prompt 的问题

### ⌨️ 统一终端输入回显策略
- Web 终端不再本地回显输入
- CLI 端不再把本地键盘输入单独广播到浏览器
- 统一以 PTY/ConPTY 的真实输出作为浏览器渲染来源

### 🔨 统一多平台构建脚本
- `build-all.sh` 现在直接构建全部受支持客户端平台
- Windows 客户端已纳入 `build-all.sh`（amd64 + arm64）
- `build-windows.sh` 专注于 Windows 产物，`Makefile` 目标同步对齐
